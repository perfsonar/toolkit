Chapter 1. Requirements

Cacti requires that the following software is installed on your system.

  • RRDTool 1.0.49 or greater, 1.4+ recommended

  • MySQL 5.x or greater

  • PHP 5.1 or greater

  • Web Server that supports PHP e.g. Apache or IIS
